# Moscow Hack: SberCloud
Разработка открытой платформы для мгновенного обмена сообщениями.

[Server] Java (Spring)

[Client] React(Typescript)

[Database] PostgreSQL

[Deployment] Docker, Kubernetes


## Getting Started

```
brew install npm
```

```
Java 11, maven
```

## Build and Run

client
```
cd client
npm i
npm run start
front: http://localhost:3000
```

server
```
server: http://localhost:8080/ 
```
